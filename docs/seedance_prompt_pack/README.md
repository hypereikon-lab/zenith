# Seedance 2 Motion-Plate Prompt Method

This pack adapts the Seedance corpus language mechanics to Zenith's still-image plus 2.5D guide workflow.

The corpus suggests a practical rule: working prompts are compact direction. They bind references, name the shot, move it with concrete verbs, then add short locks.

For this workflow:

- Image1 is the visual base
- Video1 is the motion guide
- the final prompt should say that once, then describe the desired shot
- guide artifacts should be rejected with short phrases, not explained at length

Target 90-160 words. Longer prompts are useful only when the motion has clear timed beats.

The prompt should sound like production direction for Seedance, not a diagnostic report.
